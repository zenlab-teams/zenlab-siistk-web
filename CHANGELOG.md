# Riwayat Perubahan

Semua perubahan penting pada proyek ini akan dicatat di file ini.

Format mengikuti [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
dan proyek ini mengikuti [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Workflow CI GitHub Actions untuk validasi proyek.
- Terjemahan antarmuka ke bahasa Indonesia.

### Changed

- README diperbarui untuk branding TelatenKarya dan screenshot terbaru.
- Dokumentasi instalasi, fitur, dependency, dan catatan refactoring diperbarui.

## [0.3.0] - 2026-06-27

### Added

- Cakupan test fitur untuk alur autentikasi.
- Cakupan test fitur untuk manajemen produk dan customer.
- Cakupan test fitur untuk manajemen offer.
- Cakupan test fitur untuk manajemen order dan invoice.
- Cakupan test fitur untuk manajemen user.

### Fixed

- Kompatibilitas query dashboard dengan SQLite.

### Removed

- Default Laravel example tests.

## [0.2.0] - 2026-06-19

### Added

- Dukungan lokasi berbasis peta untuk offer.
- Threshold stok minimum opsional untuk produk.
- Dokumentasi proyek untuk instalasi, fitur, dependency, catatan refactoring, dan GitHub Actions.
- Screenshot proyek.
- Catatan change request untuk perilaku stok minimum opsional.

### Changed

- Informasi tim di README dan screenshot proyek.
- Perilaku stok produk untuk mendukung threshold stok minimum opsional.

## [0.1.0] - 2026-06-02

### Added

- Alur input massal produk.
- Validasi backend dan route untuk pembuatan massal produk.
